# React Todo App

This project is a Todo App in React JS Framework.

## Usage: 

```
git clone https://github.com/VirajRG/todo-app.git
cd todo-app
git remote set-url origin [url of new project]
npm install
npm run sass
npm start
```

#### or just 
